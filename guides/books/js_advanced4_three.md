title: 操作符

其中包括**数学操作符** **位操作符** **关系操作符** **相等操作符**

### ++
前缀
```js
 let age = 12
    let qianzhuiAge = ++age
    // let shouzhuiAge =  age++ // 
    console.log(age) // 13 
    console.log(qianzhuiAge)  // 13
```
```js
 let age = 12
    // let qianzhuiAge = ++age
    let houzhuiAge =  age++ // 
    console.log(age) // 13 
    console.log(houzhuiAge)  // 12
```
